# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-30

### Added
- Initial release of NYSDS Override Extension Template
- Manifest V3 Chrome extension structure
- Toggle on/off functionality with persistent state via `chrome.storage.local`
- NYSDS fonts and tokens integration via CDN
- Reversible DOM modification system in `override.js`
- Popup UI with visual toggle indicator
- Comprehensive README with installation and usage instructions
- Placeholder icons (16px, 48px, 128px)
- Example CSS override patterns for typography, buttons, forms, and links
