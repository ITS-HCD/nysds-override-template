# NYSDS Fonts

This directory contains the NYS Design System fonts bundled locally for the extension.

## Required Font Files

Download the **Full** font bundle from the private repo and add these files:

- `proximanova-regular.woff2`
- `proximanova-regular.woff`
- `dsari-bold.woff2`
- `dsari-bold.woff`
- `oswald-variable.woff2`
- `oswald-variable.woff`

## Download Instructions

1. Go to [ITS-HCD/nysds-fonts](https://github.com/ITS-HCD/nysds-fonts) (internal access required)
2. Download the **Full Fonts** bundle
3. Extract and copy the `.woff` and `.woff2` files to this directory
